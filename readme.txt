== Description ==
WP Post Updated will allow your subscriber to receive a mail when an post is updated on your website.
In the back-oofice, Admin can setup the post Type to follow

== Installation ==
1- unzip
2- upload to wp-content/plugin
3- Go to your dashboard to activate it
4- have fun!
